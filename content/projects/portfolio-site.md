## Overview

Ashewyna is a full-stack event organizing platform built to coordinate organizers, vendors, and customers in a single workflow.

## Technical Details

- Built the frontend in Flutter for a responsive, mobile-first user experience.
- Implemented backend services with Node.js and MongoDB.
- Led API development for authentication, payment processing, and real-time updates.
- Collaborated with a team to ship user-friendly features and stable deployments.
