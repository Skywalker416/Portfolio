## Overview

Qignit is a mobile music app designed for Amharic content discovery and playback with performance and usability as core priorities.

## Technical Details

- Designed and developed the mobile app using Flutter.
- Built an intuitive interface with smooth audio playback features.
- Integrated cloud storage and APIs for streaming and content management.
- Optimized performance for offline playback and lower data consumption.
